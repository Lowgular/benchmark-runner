# Constraint: signal-type

## Intent fragment

Expand `PropertyDeclaration` with:

### signal property:

- has type with resolved declaration name in `Signal`, `WritableSignal`, `InputSignal`, `ModelSignal` and is imported from `@angular/core`

### not signal property:

- does not have type with resolved declaration name in `Signal`, `WritableSignal`, `InputSignal`, `ModelSignal` and is imported from `@angular/core`
